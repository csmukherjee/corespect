# CoreSPECT

This repository contains the codes for CoreSPECTED-K-Means and CoreSPECTED-HDBSCAN, as implemented in [1]. These present improved variants of K-Means and HDBDCAN by exploiting s multi-layered structure in data.

Download the dependency list by `pip install -r requirements.txt`

Note that you may not be able to run numba with python 3.12. Please use Python 3.9-3.11 for best compatibility.
   
Check test_run.ipynb for more details.

Checkout https://github.com/csmukherjee/cplearn for a more comprehensive set of algorithms that follow and improve on the CoreSPECT framework of [1]!  

[1] Chandra Sekhar Mukherjee, Joonyoung Bae, and Jiapeng Zhang. "CoreSPECT: Enhancing Clustering Algorithms via an Interplay of Density and Geometry." In The 29th International Conference on Artificial Intelligence and Statistics (AISTATS). 2026.
